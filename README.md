# 📄 Документация проекта DOCX Parser для спецификаций

## 1. Назначение

Этот проект предназначен для автоматизации процесса извлечения табличной информации из документов формата **DOCX** (Microsoft Word), преобразования её в удобный формат **JSON**, а также последующего экспорта в файл формата **Excel**. 

С помощью данного инструмента вы сможете легко извлекать данные спецификаций, структурировать их и экспортировать в таблицы с удобными стилями оформления.

## 2. Установка и зависимости

Для работы проекта необходимы следующие библиотеки:

```bash
pip install python-docx openpyxl json
```

## 3. Структура проекта

Структура директорий и файлов выглядит следующим образом:

```
./docx_parser/
│   ├── docx_table_parser.py     # Парсер таблиц из DOCX
│   ├── excel_writer.py          # Класс для записи в Excel
│   ├── word_reader.py           # Чтение текста и таблиц из DOCX
│   └── main.py                  # Основной скрипт запуска программы
```

## 4. Использование (как запустить)

### Запуск скрипта

Запустите основной скрипт `main.py` командой:

```bash
python main.py 
```

По умолчанию результат будет сохранён в папку `output_excel`

### Пример вывода

После выполнения программа выведет сообщение вида:

```
Таблицы успешно извлечены!
Файл спецификации сохранён в формате Excel.
```

## 5. Формат входных файлов

Формат входных файлов должен соответствовать стандарту Microsoft Word (*.docx*). В документе обязательно наличие хотя бы одной таблицы, содержащей столбцы с названиями:

| Поз. | Наименование        | Тип       | Кол-во | Масса |
|------|----------------------|-----------|--------|-------|
| 1    | Труба стальная      | Металл    | 10 шт. | 5 кг  |
| 2    | Болт М8              | Крепёжный | 50 шт. | 0,5 кг|

## 6. Пример результата

Пример структуры файла после обработки:

```
[
  {
    "Поз.": "1",
    "Наименование": "Труба стальная",
    "Тип": "Металл",
    "Кол-во": "10 шт.",
    "Масса": "5 кг"
  },
  {
    "Поз.": "2",
    "Наименование": "Болт М8",
    "Тип": "Крепёжный",
    "Кол-во": "50 шт.",
    "Масса": "0,5 кг"
  }
]
```

Результатом является корректная таблица в формате Excel с автоматически настроенными стилями и шириной колонок.
